# Учебная практика
Игра написана на языке программирования С# на движке Unity в рамках учебной практики с Санкт-Петербургском Государственном Университете Аэрокосмического Приборостроения (ГУАП) студентом группы 4526 1 курса направления 09.03.02 (информационные системы и технологии в дизайне) Ушаковой Полиной.
# Суть игры

Нужно расследовать отравление в ресторане. Осмотрите три блюда,
соберите улики, допросите подозреваемых и выберите убийцу на доске расследования.
Будьте внимательны: ложный след может запутать.

# Управление

WASD/стрелки — движение
E — осмотр и диалог
Q — доска расследования
Пробел — следующая реплика
M — звук
